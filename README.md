# DBT_ADDRESS

住所データの変換・正規化パイプライン。BigQuery上で動作するDBTもどき。

非公開。問い合わせは森口大輔まで。

---

## 概要

`logistics-449115` プロジェクト内の住所データを変換・正規化する。
`geocoding-engine` と連携して住所→緯度経度変換も担う。

---

## 設計思想

Cloud Run上の中継エンジンとして機能。
フロントエンドなし — 他のサービスやスケジューラから叩かれることを想定。

住所データの変換ロジックを一箇所に集約することで、
複数サービス（店舗マップ、コース分析、スコアリング等）が同じ正規化済みデータを使える。
